        <h3 class="subheader section-title-left">
          Requirements
        </h3>
        <span class="section-title-line"></span>
      </div>

      <div>
        <h4>What to Build</h4>
<p>Build a no/low-code conversational AI agent using <a href="https://vertexaiconversation.cloud.google.com/" target="_blank" rel="noopener">Vertex AI Agent Builder</a> that falls into one of four categories:&nbsp;</p>
<ul>
<li>
<p>Knowledge Bot (for example: skills up or educates a persona that you specify)</p>
</li>
<li>

      Judging Criteria
    </h3>
    <span class="section-title-line"></span>
  </div>

  <ul class="no-bullet">
      <li>
        <strong>Technological Implementation </strong><br>
        Does the project demonstrate quality software development? Does the project thoroughly leverage the generative AI agents? How is the quality of the code? How precise and clever are your Goal and Instructions prompts used to train the agent?
      </li>
      <li>
        <strong>Potential Impact</strong><br>
        How big of an impact could the project have on the target persona(s)? How big of an impact could it have beyond the target community?
      </li>
      <li>
        <strong>Utility</strong><br>
        Are you solving an existing problem? Is it an application that would encourage a user to use it more than once for its benefits?
      </li>
      <li>

        Impress the judges. Does it use newly released features? Does it have a quality that stands out creatively?
